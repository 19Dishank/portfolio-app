## 🪄 About the Project

A **next-generation developer portfolio** built to blend smooth animations, minimal design, and responsive performance.  
It’s not just a showcase — it’s an **experience**.

---

## ✨ Features

- 🎬 Framer Motion–powered transitions
- 💻 Fully responsive for all devices
- 🖼️ Interactive image hover-zoom in project popups
- 🌗 Dark mode optimized
- 🎨 Modern design — clean, smooth, and aesthetic

---

## 🛠️ Tech Stack

**Core:**

- ⚛️ React.js
- 💅 CSS3
- ⚙️ JavaScript (ES6+)

**Enhancements:**

- 🎞️ Framer Motion (for buttery-smooth transitions)
- 🧩 shadcn/ui (for clean, accessible UI components)
- 🧠 Lucide Icons (for sleek, modern icons)

## ⚙️ Installation

```bash
# Clone the repo (read-only)
git clone https://github.com/19Dishank/portfolio-app

# Move into the folder
cd portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```
